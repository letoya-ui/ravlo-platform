Style Guide



The Style Guide defines the visual rules and identity that all Ravlo patterns must follow. It ensures that every pattern — hero, dashboard, flow, feature block, content, and marketing — feels unified, cinematic, and unmistakably Ravlo.



Ravlo’s pattern style language is:

\- architectural

\- disciplined

\- cinematic

\- calm

\- premium



1\. Purpose

The Style Guide provides the visual foundation for all pattern structures. It ensures that patterns share a consistent rhythm, tone, and identity across the entire Ravlo ecosystem.



The style guide defines:

\- color usage within patterns

\- typography hierarchy

\- spacing and rhythm

\- radii and shape language

\- iconography rules

\- motion and transitions

\- tone and visual identity



2\. Core Principles

All patterns follow Ravlo’s four guiding principles:



Blueprint Alignment

Patterns must use disciplined grid structure and precise alignment.



Cinematic Spacing

Patterns must include generous breathing room and intentional negative space.



Clarity Over Cleverness

Patterns must prioritize readability and comprehension over decoration.



Emotional Safety

Patterns must feel calm, grounded, and predictable.



3\. Color Usage

Patterns use a dark‑mode‑first palette inspired by architectural materials and cinematic lighting.



Color rules:

\- primary colors for key actions and highlights

\- secondary colors for accents and supporting elements

\- semantic colors for states (success, warning, error)

\- surface colors for depth and hierarchy



4\. Typography

Patterns use a strict, predictable typographic hierarchy.



Typography rules:

\- consistent heading scale

\- readable body text sizes

\- controlled line lengths

\- disciplined spacing between text blocks

\- limited weight variations for clarity



5\. Spacing \& Rhythm

Patterns rely on token‑driven spacing to create predictable rhythm.



Spacing rules:

\- consistent vertical rhythm

\- grid‑aligned horizontal spacing

\- structured section spacing

\- balanced padding within blocks



6\. Radii \& Shape Language

Patterns use subtle, architectural radii that feel premium and intentional.



Radii rules:

\- consistent corner radius across cards, panels, and blocks

\- minimal curvature to maintain architectural identity



7\. Iconography

Icons used within patterns must be minimal, geometric, and architectural.



Icon rules:

\- consistent stroke weight

\- balanced proportions

\- clear alignment with text and grid



8\. Motion \& Transitions

Motion within patterns is subtle and purposeful.



Motion rules:

\- soft fades and slides

\- consistent timing and easing

\- motion used only to reinforce clarity, never decoration



9\. Tone \& Identity

Patterns must express Ravlo’s core identity:

\- confident

\- calm

\- structured

\- investor‑grade



The Style Guide ensures that every pattern — regardless of category — feels unified, intentional, and unmistakably Ravlo.
