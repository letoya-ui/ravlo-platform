# \# AI Partner Matching

# 

# AI Partner Matching recommends the best partners for each deal or service request.

# 

# \## Purpose

# 

# This step answers:

# “Who is the best partner for this job?”

# 

# It provides:

# \- match score  

# \- reasoning  

# \- availability  

# \- performance indicators  

# 

# \## Inputs

# 

# \- deal type  

# \- location  

# \- partner role  

# \- partner performance  

# \- partner availability  

# \- investor preferences  

# 

# \## Anatomy

# 

# \- Match card  

# \- Score  

# \- Explanation  

# \- CTA to request service  

# 

# \## Behavior

# 

# \- AI updates matches dynamically.  

# \- Users can override AI suggestions.  

# 

# \## Do

# 

# \- Keep explanations short and clear.  

# \- Prioritize transparency.  

# 

# \## Don’t

# 

# \- Present AI matches as mandatory.

#
