# \# Partner Roles

# 

# Partner Roles define the categories of professionals available in the marketplace.

# 

# \## Purpose

# 

# This step answers:

# “What types of partners can I work with?”

# 

# It provides:

# \- clear role definitions  

# \- expectations for each role  

# \- service categories  

# \- partner capabilities  

# 

# \## Core Roles

# 

# \- Realtor  

# \- Contractor  

# \- Inspector  

# \- Appraiser  

# \- Lender  

# \- Title / Attorney  

# \- Insurance  

# \- Property Manager  

# \- Designer (optional future role)  

# 

# \## Anatomy

# 

# Each role includes:

# \- Name  

# \- Description  

# \- Services offered  

# \- Required credentials  

# \- Typical workflows  

# 

# \## Do

# 

# \- Keep role definitions simple and clear.  

# \- Align roles with real investor workflows.  

# 

# \## Don’t

# 

# \- Create overlapping or confusing roles.

#
