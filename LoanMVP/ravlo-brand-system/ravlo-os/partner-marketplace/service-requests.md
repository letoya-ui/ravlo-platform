# \# Service Requests

# 

# Service Requests allow investors to request quotes, bids, or services from partners.

# 

# \## Purpose

# 

# This step answers:

# “How do I engage this partner?”

# 

# It provides:

# \- request form  

# \- messaging  

# \- file uploads  

# \- status tracking  

# 

# \## Request Types

# 

# \- Contractor bid  

# \- Realtor CMA  

# \- Inspection  

# \- Appraisal  

# \- Title / insurance quote  

# \- Property management onboarding  

# 

# \## Anatomy

# 

# \- Request form  

# \- Deal or property selector  

# \- Message box  

# \- File upload  

# \- Status timeline  

# 

# \## Behavior

# 

# \- Partners receive notifications.  

# \- Investors can track progress.  

# \- Requests integrate with partner dashboards.  

# 

# \## Do

# 

# \- Keep forms short and focused.  

# \- Allow attaching deal context.  

# 

# \## Don’t

# 

# \- Require unnecessary details upfront.

#
