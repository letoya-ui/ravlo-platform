# \# Partner CRM

# 

# Partner CRM is the admin‑side tool for managing partner onboarding, approvals, and performance.

# 

# \## Purpose

# 

# This step answers:

# “How do we manage partners at scale?”

# 

# It provides:

# \- partner approvals  

# \- partner tiers  

# \- performance metrics  

# \- lead routing  

# \- billing (future)  

# 

# \## Anatomy

# 

# \- Partner list  

# \- Status indicators  

# \- Tier badges  

# \- Performance metrics  

# \- Admin actions  

# 

# \## Behavior

# 

# \- Admins can approve or reject partners.  

# \- CRM integrates with partner dashboards.  

# \- AI flags low‑performing partners.  

# 

# \## Do

# 

# \- Keep CRM controls clear and safe.  

# \- Use consistent status indicators.  

# 

# \## Don’t

# 

# \- Allow destructive actions without confirmation.
