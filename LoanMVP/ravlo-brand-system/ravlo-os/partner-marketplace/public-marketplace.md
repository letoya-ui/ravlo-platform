# \# Public Marketplace

# 

# The Public Marketplace is the front‑facing directory where investors and borrowers discover partners.

# 

# \## Purpose

# 

# This step answers:

# “Who is available in my area?”

# 

# It provides:

# \- partner search  

# \- filters (role, location, specialty)  

# \- partner cards  

# \- limited public profiles  

# 

# \## Anatomy

# 

# \- Search bar  

# \- Filter bar  

# \- Partner grid  

# \- Pagination or infinite scroll  

# \- Public profile preview  

# 

# \## Behavior

# 

# \- Public profiles show limited information.  

# \- Full profiles require login.  

# \- Service requests require authentication.  

# 

# \## Do

# 

# \- Keep partner cards clean and scannable.  

# \- Use clear filters.  

# 

# \## Don’t

# 

# \- Show sensitive partner data publicly.
