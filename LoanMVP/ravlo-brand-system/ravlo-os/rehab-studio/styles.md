# \# Style Presets

# 

# Style Presets define the visual direction of the renovation. They influence variations, AI rehab plans, and cost ranges.

# 

# \## Purpose

# 

# This step answers:

# “What design direction should this renovation follow?”

# 

# It provides:

# \- curated style options  

# \- visual previews  

# \- cost implications  

# \- AI generation context  

# 

# \## Example Presets

# 

# \- Modern  

# \- Contemporary  

# \- Farmhouse  

# \- Minimalist  

# \- Transitional  

# \- Industrial  

# 

# \## Anatomy

# 

# Each preset includes:

# \- Name  

# \- Description  

# \- Visual preview  

# \- Cost implication  

# \- AI tags  

# 

# \## Behavior

# 

# \- Selecting a preset updates variation generation.  

# \- AI rehab plan adapts to the selected style.  

# 

# \## Do

# 

# \- Keep presets visually distinct.  

# \- Provide clear cost expectations.  

# 

# \## Don’t

# 

# \- Mix incompatible materials or styles.  

# \- Hide what’s included in each preset.

#
