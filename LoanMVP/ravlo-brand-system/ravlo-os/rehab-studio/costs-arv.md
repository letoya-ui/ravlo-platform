# \# Costs \& ARV

# 

# Costs \& ARV estimate the financial impact of the renovation, including total rehab cost and projected after‑repair value.

# 

# \## Purpose

# 

# This step answers:

# “What will this renovation cost, and what will it be worth?”

# 

# It provides:

# \- total rehab cost  

# \- cost ranges  

# \- ARV estimate  

# \- ROI implications  

# \- risk indicators  

# 

# \## Anatomy

# 

# \- Cost breakdown  

# \- ARV range  

# \- ROI metrics  

# \- Trend indicators  

# \- Notes and assumptions  

# 

# \## Behavior

# 

# \- Costs update based on AI rehab plan.  

# \- ARV updates based on comps + style impact.  

# 

# \## Do

# 

# \- Show ranges, not single numbers.  

# \- Make assumptions transparent.  

# 

# \## Don’t

# 

# \- Present unrealistic or overly precise estimates.  

# \- Hide soft costs or contingency.
