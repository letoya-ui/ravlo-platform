# \# Rehab Studio

# 

# Rehab Studio transforms raw property conditions into investor‑ready renovation concepts. It blends before‑images, style presets, variations, AI rehab plans, and cost/ARV logic into a single cinematic workspace.

# 

# \## Purpose

# 

# Rehab Studio answers:

# “What will this renovation look like, and what will it cost?”

# 

# It provides:

# \- before‑image upload  

# \- style preset selection  

# \- variation generation  

# \- AI rehab plan  

# \- cost + ARV logic  

# \- investor‑ready outputs  

# 

# \## Workflow Stages

# 

# 1\. Before Image  

# 2\. Style Presets  

# 3\. Variations  

# 4\. AI Rehab Plan  

# 5\. Costs \& ARV  

# 6\. Outputs  

# 

# \## Component References

# 

# \- Image upload  

# \- Style cards  

# \- Variation grid  

# \- AI text block  

# \- Cost tables  

# \- Before/after slider  

# \- Output preview frames  

# 

# \## Do

# 

# \- Keep the experience visual and cinematic.  

# \- Use clear, investor‑friendly language.  

# \- Make AI outputs easy to understand and act on.  

# 

# \## Don’t

# 

# \- Overload the user with technical renovation jargon.  

# \- Hide assumptions behind cost or ARV calculations.
