# \# Rehab Outputs

# 

# Rehab Outputs provide investor‑ready visuals and summaries for presentations, partners, and lenders.

# 

# \## Purpose

# 

# This step answers:

# “What can I share with my team or lender?”

# 

# It provides:

# \- before/after slider  

# \- featured variation  

# \- AI rehab plan summary  

# \- cost + ARV summary  

# \- exportable presentation sheet  

# 

# \## Anatomy

# 

# \- Before/after slider  

# \- Featured variation  

# \- Summary metrics  

# \- Download/export actions  

# 

# \## Behavior

# 

# \- Outputs update when variations or rehab plan change.  

# \- Users can save multiple versions.  

# 

# \## Do

# 

# \- Keep previews large and cinematic.  

# \- Use consistent framing and aspect ratios.  

# 

# \## Don’t

# 

# \- Mix outdated and updated outputs without versioning.  

# \- Hide generation details or assumptions.
