# \# Before Image

# 

# The Before Image is the foundation of Rehab Studio. It anchors the renovation concept and provides context for AI generation.

# 

# \## Purpose

# 

# This step answers:

# “What does the property look like today?”

# 

# It provides:

# \- visual baseline  

# \- context for style presets  

# \- context for AI rehab plan  

# \- context for cost estimation  

# 

# \## Anatomy

# 

# \- Image upload area  

# \- Drag‑and‑drop support  

# \- File validation  

# \- Optional cropping  

# \- Optional room tagging  

# 

# \## Behavior

# 

# \- AI analyzes the image for condition, materials, and potential issues.  

# \- Users can upload multiple images (future enhancement).  

# 

# \## Do

# 

# \- Keep the upload area large and inviting.  

# \- Support common image formats.  

# 

# \## Don’t

# 

# \- Force users to crop unless necessary.  

# \- Reject images without clear explanation.
