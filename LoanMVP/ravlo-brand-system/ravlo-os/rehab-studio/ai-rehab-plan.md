# \# AI Rehab Plan

# 

# The AI Rehab Plan converts the selected variation into a structured renovation scope.

# 

# \## Purpose

# 

# This step answers:

# “What exactly needs to be done to achieve this design?”

# 

# It provides:

# \- line‑item scope  

# \- material recommendations  

# \- labor assumptions  

# \- risk indicators  

# \- timeline estimates  

# 

# \## Anatomy

# 

# \- AI summary block  

# \- Scope categories  

# \- Line items  

# \- Cost ranges  

# \- Risk flags  

# 

# \## Behavior

# 

# \- AI uses the before image + selected variation + style preset.  

# \- Users can edit or override items.  

# \- Totals update dynamically.  

# 

# \## Do

# 

# \- Keep the plan clear and actionable.  

# \- Group items logically.  

# 

# \## Don’t

# 

# \- Hide assumptions behind cost or timeline.  

# \- Mix unrelated tasks in one line item.
