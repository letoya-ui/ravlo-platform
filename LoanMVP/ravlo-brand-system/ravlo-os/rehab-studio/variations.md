# \# Variations

# 

# Variations are AI‑generated design options based on the before image and selected style preset.

# 

# \## Purpose

# 

# This step answers:

# “What are the possible design directions for this renovation?”

# 

# It provides:

# \- multiple visual concepts  

# \- style‑aligned variations  

# \- investor‑friendly comparison  

# 

# \## Anatomy

# 

# \- Variation grid  

# \- Thumbnail previews  

# \- Full‑size preview modal  

# \- “Select as Featured” action  

# 

# \## Behavior

# 

# \- AI generates 2–6 variations per style.  

# \- Users can regenerate or refine variations.  

# \- Selected variation influences the AI rehab plan.  

# 

# \## Do

# 

# \- Keep variation previews consistent in framing.  

# \- Allow users to compare variations easily.  

# 

# \## Don’t

# 

# \- Generate variations with inconsistent lighting or angles.  

# \- Overwhelm users with too many options.
