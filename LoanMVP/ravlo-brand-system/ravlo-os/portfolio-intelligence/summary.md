# \# Portfolio Summary

# 

# Portfolio Summary provides a complete, investor‑ready overview of the portfolio.

# 

# \## Purpose

# 

# This step answers:

# “What is the state of my portfolio today?”

# 

# It provides:

# \- key metrics  

# \- performance summary  

# \- risk summary  

# \- cash flow summary  

# \- market context  

# \- AI recommendations  

# 

# \## Anatomy

# 

# \- Summary metrics  

# \- Trend highlights  

# \- Risk indicators  

# \- Cash flow overview  

# \- Recommended next steps  

# 

# \## Behavior

# 

# \- Summary updates dynamically.  

# \- Users can export or share the summary.  

# 

# \## Do

# 

# \- Keep the summary clean and scannable.  

# \- Highlight the most important metrics.  

# 

# \## Don’t

# 

# \- Overload the summary with raw data.
