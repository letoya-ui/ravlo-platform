# \# Risk \& Exposure

# 

# Risk \& Exposure identifies vulnerabilities in the portfolio.

# 

# \## Purpose

# 

# This step answers:

# “What risks exist in my portfolio?”

# 

# It provides:

# \- loan exposure  

# \- leverage ratios  

# \- vacancy risk  

# \- market risk  

# \- rehab risk  

# \- liquidity risk  

# 

# \## Anatomy

# 

# \- Risk score  

# \- Exposure breakdown  

# \- Heatmap or risk matrix  

# \- Recommended actions  

# 

# \## Behavior

# 

# \- AI flags high‑risk properties or deals.  

# \- Risk score updates dynamically.  

# 

# \## Do

# 

# \- Keep risk indicators clear and non‑alarming.  

# \- Provide actionable recommendations.  

# 

# \## Don’t

# 

# \- Overwhelm users with technical risk terminology.
