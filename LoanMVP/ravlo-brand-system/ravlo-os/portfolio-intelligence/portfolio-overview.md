# \# Portfolio Overview

# 

# Portfolio Overview provides a high‑level snapshot of the investor’s entire portfolio.

# 

# \## Purpose

# 

# This step answers:

# “What does my portfolio look like right now?”

# 

# It provides:

# \- total portfolio value  

# \- total equity  

# \- total debt  

# \- number of properties  

# \- number of active deals  

# \- cash position  

# 

# \## Anatomy

# 

# \- KPI row  

# \- Trend indicators  

# \- Portfolio composition chart  

# \- Quick links to properties and deals  

# 

# \## Behavior

# 

# \- Updates dynamically as deals and properties change.  

# \- AI highlights major shifts or anomalies.  

# 

# \## Do

# 

# \- Keep KPIs clean and scannable.  

# \- Use consistent formatting for currency and percentages.  

# 

# \## Don’t

# 

# \- Mix property‑level and portfolio‑level metrics without clarity.
