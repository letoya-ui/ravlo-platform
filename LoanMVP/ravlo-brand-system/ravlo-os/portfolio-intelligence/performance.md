# \# Performance Metrics

# 

# Performance Metrics track how the portfolio is performing over time.

# 

# \## Purpose

# 

# This step answers:

# “How is my portfolio trending?”

# 

# It provides:

# \- ROI  

# \- cash‑on‑cash return  

# \- appreciation  

# \- rental yield  

# \- occupancy  

# \- expense ratios  

# 

# \## Anatomy

# 

# \- Trend charts  

# \- Metric cards  

# \- Comparison vs previous periods  

# \- Notes and assumptions  

# 

# \## Behavior

# 

# \- Charts update based on selected time range.  

# \- AI highlights performance anomalies.  

# 

# \## Do

# 

# \- Use clear, readable charts.  

# \- Provide context for each metric.  

# 

# \## Don’t

# 

# \- Overcomplicate with too many time ranges.
