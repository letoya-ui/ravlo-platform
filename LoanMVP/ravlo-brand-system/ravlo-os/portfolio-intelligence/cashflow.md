# \# Cash Flow

# 

# Cash Flow models income, expenses, and net cash flow across the portfolio.

# 

# \## Purpose

# 

# This step answers:

# “How much cash is my portfolio generating?”

# 

# It provides:

# \- monthly cash flow  

# \- annual cash flow  

# \- income vs expenses  

# \- projections  

# \- sensitivity analysis  

# 

# \## Anatomy

# 

# \- Cash flow chart  

# \- Income breakdown  

# \- Expense breakdown  

# \- Net cash flow metric  

# 

# \## Behavior

# 

# \- Updates based on property performance and financing.  

# \- AI highlights cash flow risks or opportunities.  

# 

# \## Do

# 

# \- Keep charts simple and readable.  

# \- Show both actuals and projections.  

# 

# \## Don’t

# 

# \- Hide assumptions behind projections.

#
