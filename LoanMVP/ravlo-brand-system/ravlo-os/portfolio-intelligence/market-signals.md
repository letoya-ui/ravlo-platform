# \# Market Signals

# 

# Market Signals provide external context that affects portfolio performance.

# 

# \## Purpose

# 

# This step answers:

# “What’s happening in the market that affects my portfolio?”

# 

# It provides:

# \- interest rate trends  

# \- local market trends  

# \- rent trends  

# \- price trends  

# \- supply/demand indicators  

# 

# \## Anatomy

# 

# \- Signal cards  

# \- Trend charts  

# \- Geographic indicators  

# \- AI commentary  

# 

# \## Behavior

# 

# \- Signals update based on investor’s markets.  

# \- AI highlights relevant changes.  

# 

# \## Do

# 

# \- Keep signals relevant and localized.  

# \- Use clear trend indicators.  

# 

# \## Don’t

# 

# \- Overload users with national‑level noise.
