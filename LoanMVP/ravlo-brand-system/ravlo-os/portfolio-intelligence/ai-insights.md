# \# AI Insights

# 

# AI Insights synthesize all portfolio data into prioritized recommendations.

# 

# \## Purpose

# 

# This step answers:

# “What should I do next?”

# 

# It provides:

# \- risk alerts  

# \- opportunity alerts  

# \- performance insights  

# \- market‑driven recommendations  

# \- next steps  

# 

# \## Anatomy

# 

# \- Insight cards  

# \- Impact level  

# \- Explanation  

# \- Recommended action  

# 

# \## Behavior

# 

# \- Insights update dynamically.  

# \- Users can dismiss or snooze insights.  

# 

# \## Do

# 

# \- Keep insights concise and actionable.  

# \- Prioritize by impact.  

# 

# \## Don’t

# 

# \- Present conflicting or redundant insights.
