# \# Hero Panel

# 

# The Hero Panel sets the emotional tone for the Command Center. It welcomes the investor, surfaces an AI‑generated summary, and provides a quick sense of how their portfolio is performing.

# 

# \## Purpose

# 

# The Hero Panel answers:  

# “What changed since I last checked Ravlo?”

# 

# It should:

# \- greet the investor by name  

# \- summarize key changes and highlights  

# \- surface one or two recommended next actions  

# 

# \## Anatomy

# 

# \- Greeting (e.g., “Good morning, Jayden”)  

# \- Subheadline (e.g., “Your portfolio is stable. 2 deals need attention today.”)  

# \- AI summary block (short paragraph or bullet list)  

# \- Optional: key trend indicator (e.g., portfolio up/down vs last period)  

# \- Optional: primary call‑to‑action (e.g., “Review today’s priorities”)  

# 

# \## Layout

# 

# \- Positioned at the top of the Command Center.  

# \- Full‑width or spanning the main content column.  

# \- Uses a cinematic but restrained background (Midnight, Blueprint, or subtle gradient) that aligns with the Ravlo visual OS.  

# 

# \## Component References

# 

# \- Page hero / banner  

# \- Text blocks (headline, subheadline, body)  

# \- AI insight block  

# \- Primary button  

# \- Optional iconography  

# 

# \## Behavior

# 

# \- AI summary should update based on recent activity, alerts, and portfolio changes.  

# \- The primary CTA should deep‑link into the most relevant module (e.g., Deal Architect, Portfolio Intelligence, Alerts \& Tasks).  

# 

# \## States

# 

# \- \*\*Normal:\*\* Calm, informative summary.  

# \- \*\*Opportunity‑focused:\*\* When there are strong opportunities (e.g., “3 high‑ROI deals identified”).  

# \- \*\*Risk‑focused:\*\* When there are critical issues (e.g., “1 loan at risk, 2 overdue tasks”).  

# \- \*\*Empty state:\*\* For new users, show onboarding guidance and a simple explanation of what the Hero Panel will show once data is connected.  

# 

# \## Do

# 

# \- Keep the summary short and human‑readable.  

# \- Use language that feels advisory, not alarmist.  

# \- Make the primary CTA meaningful and contextual.  

# 

# \## Don’t

# 

# \- Overload the Hero Panel with dense metrics.  

# \- Use long paragraphs that are hard to scan.  

# \- Show conflicting or redundant information already visible elsewhere on the page.
