# \# Table Patterns

# 

# Table Patterns define how Ravlo displays structured data.

# 

# \## Purpose

# 

# Tables answer:

# “What are the key items I need to compare?”

# 

# They provide:

# \- sortable columns  

# \- filters  

# \- row actions  

# \- status indicators  

# 

# \## Anatomy

# 

# \- Header row  

# \- Data rows  

# \- Status badges  

# \- Row actions  

# \- Pagination  

# 

# \## Behavior

# 

# \- Columns should be sortable.  

# \- Filters should be simple and clear.  

# 

# \## Do

# 

# \- Keep tables clean and readable.  

# \- Use consistent column spacing.  

# 

# \## Don’t

# 

# \- Overload tables with too many columns.

#
