# \# Panel Patterns

# 

# Panels are the primary content containers in Ravlo OS.

# 

# \## Purpose

# 

# Panels answer:

# “What surface holds this content?”

# 

# They provide:

# \- structure  

# \- depth  

# \- spacing  

# \- grouping  

# 

# \## Anatomy

# 

# \- Steel surface  

# \- Inner shadow  

# \- Outer glow  

# \- 12–24px padding  

# 

# \## Do

# 

# \- Use panels to group related content.  

# \- Maintain consistent spacing.  

# 

# \## Don’t

# 

# \- Nest panels unnecessarily.
