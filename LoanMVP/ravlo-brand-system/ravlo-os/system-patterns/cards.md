# \# Card Patterns

# 

# Card Patterns define the modular blocks used across dashboards and detail views.

# 

# \## Purpose

# 

# Cards answer:

# “What is the smallest unit of structured information?”

# 

# They provide:

# \- metric cards  

# \- insight cards  

# \- partner cards  

# \- deal cards  

# \- property cards  

# 

# \## Anatomy

# 

# \- Header  

# \- Body  

# \- Footer (optional)  

# \- Status or badge (optional)  

# 

# \## Do

# 

# \- Keep cards minimal and clean.  

# \- Use consistent padding.  

# 

# \## Don’t

# 

# \- Over‑decorate cards.
