# \# Modal Patterns

# 

# Modals provide focused, interruptive surfaces for critical tasks.

# 

# \## Purpose

# 

# Modals answer:

# “What requires immediate attention?”

# 

# They provide:

# \- forms  

# \- confirmations  

# \- warnings  

# \- quick actions  

# 

# \## Anatomy

# 

# \- Header  

# \- Body  

# \- Footer with actions  

# 

# \## Behavior

# 

# \- Modals should be dismissible.  

# \- Use motion to soften entry.  

# 

# \## Do

# 

# \- Keep modals focused.  

# \- Use clear primary actions.  

# 

# \## Don’t

# 

# \- Overuse modals for simple tasks.

#
