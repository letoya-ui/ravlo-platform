# \# Scope of Work

# 

# Scope of Work defines the tasks, categories, and labor required to complete the project.

# 

# \## Purpose

# 

# This step answers:

# “What exactly needs to be done?”

# 

# It provides:

# \- line‑item scope  

# \- category grouping  

# \- labor vs material breakdown  

# \- timeline implications  

# 

# \## Scope Categories

# 

# \- Interior  

# \- Exterior  

# \- Systems (HVAC, electrical, plumbing)  

# \- Structural  

# \- Landscaping  

# \- Miscellaneous  

# 

# \## Anatomy

# 

# Each scope item includes:

# \- Category  

# \- Task name  

# \- Quantity  

# \- Unit cost  

# \- Total cost  

# \- Notes  

# 

# \## Behavior

# 

# \- AI can suggest scope items based on style and materials.  

# \- Users can add, remove, or adjust items.  

# \- Totals update dynamically.  

# 

# \## Do

# 

# \- Keep scope items clear and actionable.  

# \- Group items logically.  

# 

# \## Don’t

# 

# \- Mix unrelated tasks in one line item.  

# \- Hide assumptions behind costs.
