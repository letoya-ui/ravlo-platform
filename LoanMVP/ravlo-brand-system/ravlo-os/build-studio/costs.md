# \# Cost Ranges

# 

# Cost Ranges provide estimated pricing for materials, labor, and full project execution.

# 

# \## Purpose

# 

# This step answers:

# “What will this project cost?”

# 

# It provides:

# \- material cost ranges  

# \- labor cost ranges  

# \- total project estimate  

# \- contingency recommendations  

# 

# \## Cost Types

# 

# \- Material costs  

# \- Labor costs  

# \- Soft costs  

# \- Permits  

# \- Contingency  

# \- Total project cost  

# 

# \## Anatomy

# 

# Each cost range includes:

# \- Category  

# \- Low estimate  

# \- High estimate  

# \- Notes or assumptions  

# 

# \## Behavior

# 

# \- Costs update based on style, materials, and scope.  

# \- Users can override defaults.  

# 

# \## Do

# 

# \- Show ranges, not single numbers.  

# \- Make assumptions transparent.  

# 

# \## Don’t

# 

# \- Hide soft costs or contingency.  

# \- Present unrealistic estimates.

#
