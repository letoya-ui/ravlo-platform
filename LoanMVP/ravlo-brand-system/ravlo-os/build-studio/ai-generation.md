# \# AI Generation

# 

# AI Generation produces concept renders, draft blueprints, and site plans based on selected styles, materials, presets, and scope.

# 

# \## Purpose

# 

# This step answers:

# “What will this project look like?”

# 

# It provides:

# \- concept exterior render  

# \- interior render (optional)  

# \- draft blueprint  

# \- site plan  

# \- presentation sheet  

# 

# \## Inputs to AI

# 

# \- Architectural style  

# \- Material selections  

# \- Preset  

# \- Scope of work  

# \- Lot photo (optional)  

# \- Lot size / zoning (optional)  

# 

# \## Outputs

# 

# \- Render  

# \- Blueprint  

# \- Site plan  

# \- Presentation sheet  

# 

# \## Behavior

# 

# \- AI uses style + materials + scope to shape outputs.  

# \- Users can regenerate or refine outputs.  

# 

# \## Do

# 

# \- Keep prompts structured and consistent.  

# \- Provide clear refinement options.  

# 

# \## Don’t

# 

# \- Mix conflicting style or material inputs.  

# \- Generate outputs without user confirmation.

#
