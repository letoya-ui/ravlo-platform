# \# Presets

# 

# Presets are curated combinations of styles, materials, and finishes that create a cohesive design package.

# 

# \## Purpose

# 

# This step answers:

# “What is the complete design package for this project?”

# 

# It provides:

# \- pre‑built design sets  

# \- consistent material combinations  

# \- predictable cost ranges  

# \- faster decision‑making  

# 

# \## Example Presets

# 

# \- Modern Essentials  

# \- Craftsman Warmth  

# \- Minimalist White  

# \- Industrial Loft  

# \- Contemporary Luxe  

# 

# \## Anatomy

# 

# Each preset includes:

# \- Style  

# \- Material package  

# \- Finish level  

# \- Cost range  

# \- Visual preview  

# 

# \## Behavior

# 

# \- Selecting a preset auto‑fills materials and cost ranges.  

# \- Users can customize presets after selection.  

# 

# \## Do

# 

# \- Keep presets visually distinct.  

# \- Provide clear cost expectations.  

# 

# \## Don’t

# 

# \- Create presets that mix incompatible materials.  

# \- Hide what’s included in each preset.
