# \# Materials \& Finishes

# 

# Materials \& Finishes define the tactile and visual elements of the project — flooring, cabinetry, countertops, fixtures, exterior finishes, and more.

# 

# \## Purpose

# 

# This step answers:

# “What will this project feel like?”

# 

# It provides:

# \- curated material sets  

# \- finish levels (basic, standard, premium)  

# \- compatibility with architectural styles  

# \- cost implications  

# 

# \## Material Categories

# 

# \- Flooring  

# \- Countertops  

# \- Cabinets  

# \- Fixtures  

# \- Paint  

# \- Exterior siding  

# \- Roofing  

# \- Windows \& doors  

# 

# \## Anatomy

# 

# Each material includes:

# \- Swatch or image  

# \- Name  

# \- Description  

# \- Finish level  

# \- Cost range  

# \- Style compatibility  

# 

# \## Behavior

# 

# \- Selecting materials updates cost ranges dynamically.  

# \- Materials can be grouped into “packages” for presets.  

# 

# \## Do

# 

# \- Keep swatches large and clear.  

# \- Use consistent naming conventions.  

# 

# \## Don’t

# 

# \- Mix finish levels without clarity.  

# \- Hide cost differences between materials.
