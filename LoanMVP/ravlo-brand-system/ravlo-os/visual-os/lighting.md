# \# Lighting \& Atmosphere

# 

# Lighting \& Atmosphere create the cinematic presence of Ravlo OS.

# 

# \## Purpose

# 

# This step answers:

# “How does Ravlo feel alive and dimensional?”

# 

# It provides:

# \- spotlight rules  

# \- ambient lighting  

# \- depth shadows  

# \- atmospheric haze  

# 

# \## Lighting Types

# 

# \- Key light (spotlight)  

# \- Fill light (soft ambient)  

# \- Rim light (subtle edge glow)  

# 

# \## Atmosphere

# 

# \- Soft haze  

# \- Low‑contrast gradients  

# \- Controlled bloom  

# 

# \## Do

# 

# \- Use lighting to guide hierarchy.  

# \- Keep atmosphere subtle and premium.  

# 

# \## Don’t

# 

# \- Over‑light the interface.  

# \- Use harsh shadows.
