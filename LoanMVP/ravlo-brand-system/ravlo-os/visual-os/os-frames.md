# \# OS Frames

# 

# OS Frames are the cinematic compositions used for hero visuals, marketing, and product storytelling.

# 

# \## Purpose

# 

# This step answers:

# “How do we present Ravlo OS in a cinematic way?”

# 

# It provides:

# \- hero compositions  

# \- floating panel arrangements  

# \- depth layering  

# \- lighting direction  

# \- blueprint accents  

# 

# \## Frame Types

# 

# \- Hero OS frame  

# \- Dashboard frame  

# \- Module frame  

# \- Ecosystem frame  

# \- Node ring frame  

# 

# \## Do

# 

# \- Use wide, cinematic framing.  

# \- Layer panels with depth and intention.  

# 

# \## Don’t

# 

# \- Overcrowd the frame.  

# \- Break the lighting direction.

#
