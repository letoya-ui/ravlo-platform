# \# Cinematic Style

# 

# Cinematic Style defines the atmospheric, elevated look of Ravlo OS — the hybrid of real UI + cinematic presence.

# 

# \## Purpose

# 

# This step answers:

# “How does Ravlo achieve its signature cinematic feel?”

# 

# It provides:

# \- lighting rules  

# \- depth rules  

# \- glow and bloom usage  

# \- atmospheric haze  

# \- color grading  

# 

# \## Key Elements

# 

# \- Midnight gradient backgrounds  

# \- Soft spotlighting  

# \- Subtle bloom  

# \- Depth shadows  

# \- Blueprint accents  

# \- Controlled contrast  

# 

# \## Do

# 

# \- Use lighting to guide attention.  

# \- Keep bloom subtle and premium.  

# 

# \## Don’t

# 

# \- Overuse glow or neon effects.  

# \- Flatten the UI into a generic SaaS look.
