# \# UI Panels

# 

# UI Panels are the structural backbone of Ravlo OS — the surfaces that hold content, metrics, charts, and interactions.

# 

# \## Purpose

# 

# This step answers:

# “What do Ravlo’s surfaces look like?”

# 

# It provides:

# \- panel shapes  

# \- depth rules  

# \- borders and dividers  

# \- spacing and padding  

# \- layering  

# 

# \## Anatomy

# 

# \- Steel surface  

# \- Soft inner shadow  

# \- Subtle outer glow  

# \- 12–24px padding  

# \- Rounded corners (4–8px)  

# 

# \## Behavior

# 

# \- Panels float above the background.  

# \- Panels can stack in layers.  

# \- Panels maintain consistent spacing.  

# 

# \## Do

# 

# \- Keep panels clean and minimal.  

# \- Use depth to separate content.  

# 

# \## Don’t

# 

# \- Add heavy borders.  

# \- Use inconsistent padding.
