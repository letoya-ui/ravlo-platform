# \# Foundations

# 

# Foundations define the core visual identity of Ravlo OS — color, depth, spacing, and composition.

# 

# \## Purpose

# 

# Foundations answer:

# “What are the base rules that everything sits on?”

# 

# They provide:

# \- color system  

# \- spacing scale  

# \- depth system  

# \- grid system  

# \- composition rules  

# 

# \## Color System

# 

# \- Midnight (primary background)  

# \- Steel (panel surfaces)  

# \- Blueprint Blue (accents)  

# \- Soft White (text)  

# \- Signal colors (risk, success, info)  

# 

# \## Depth System

# 

# \- Layer 0: Background  

# \- Layer 1: Panels  

# \- Layer 2: Floating elements  

# \- Layer 3: Overlays and modals  

# 

# \## Spacing Scale

# 

# \- 4 / 8 / 12 / 16 / 24 / 32 / 48  

# 

# \## Composition

# 

# \- Wide, cinematic framing  

# \- Strong horizontal alignment  

# \- Breathing room around key elements  

# 

# \## Do

# 

# \- Use spacing generously.  

# \- Keep depth consistent.  

# 

# \## Don’t

# 

# \- Collapse elements too tightly.  

# \- Break the grid without intention.
