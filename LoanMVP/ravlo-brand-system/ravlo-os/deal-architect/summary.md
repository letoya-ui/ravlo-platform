# \# Deal Summary

# 

# Deal Summary provides a complete, investor‑ready overview of the deal, including all calculations, risks, and recommendations.

# 

# \## Purpose

# 

# This step answers:  

# “What’s the final verdict?”

# 

# It provides:

# \- final ARV  

# \- total rehab cost  

# \- total cash required  

# \- ROI metrics  

# \- risk profile  

# \- AI recommendation  

# \- exportable summary  

# 

# \## Anatomy

# 

# \- Summary metrics  

# \- Deal highlights  

# \- Risk indicators  

# \- ROI breakdown  

# \- Timeline  

# \- Recommended next steps  

# 

# \## Behavior

# 

# \- Summary updates dynamically as inputs change.  

# \- Users can export or share the summary.  

# 

# \## Do

# 

# \- Keep the summary clean and scannable.  

# \- Highlight the most important metrics.  

# 

# \## Don’t

# 

# \- Overload the summary with raw data.  

# \- Hide risks or assumptions.
