# \# Deal Architect

# 

# Deal Architect is Ravlo’s flagship analysis engine. It guides investors through the full lifecycle of evaluating a deal—from strategy selection to comps, rehab, financing, AI analysis, and final summary.

# 

# \## Purpose

# 

# Deal Architect answers the core investor question:  

# “Is this a good deal, and what should I do next?”

# 

# It provides:

# \- structured deal evaluation  

# \- automated data enrichment  

# \- AI‑powered insights  

# \- clear risk and ROI projections  

# \- a repeatable, investor‑grade workflow  

# 

# \## Workflow Stages

# 

# 1\. Strategy Selection  

# 2\. Deal Input  

# 3\. Comps \& Valuation  

# 4\. Rehab \& Scope  

# 5\. Financing  

# 6\. AI Analysis  

# 7\. Deal Summary  

# 

# Each stage builds on the previous one, creating a complete, defensible investment profile.

# 

# \## Component References

# 

# \- Form inputs  

# \- Metric cards  

# \- Tables  

# \- Charts  

# \- Status indicators  

# \- AI insight blocks  

# \- Navigation steps  

# 

# \## Do

# 

# \- Keep the workflow linear and predictable.  

# \- Use clear labels and investor‑friendly language.  

# \- Surface AI insights at the right moments.  

# 

# \## Don’t

# 

# \- Overload the user with unnecessary fields.  

# \- Mix strategy‑specific logic without clarity.  

# \- Hide critical risk indicators.
