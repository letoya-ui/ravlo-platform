# \# Rehab \& Scope

# 

# Rehab \& Scope estimates renovation costs, timelines, and scope of work.

# 

# \## Purpose

# 

# This step answers:  

# “What will it cost to get this property to ARV?”

# 

# It provides:

# \- line‑item rehab estimates  

# \- material + labor breakdowns  

# \- timeline estimates  

# \- risk indicators  

# 

# \## Anatomy

# 

# \- Scope categories (interior, exterior, systems, misc)  

# \- Line items  

# \- Cost ranges  

# \- Timeline estimates  

# \- Risk flags  

# 

# \## Behavior

# 

# \- AI suggests a scope based on property details.  

# \- Users can adjust quantities and materials.  

# \- Totals update dynamically.  

# 

# \## Do

# 

# \- Keep line items grouped logically.  

# \- Show cost ranges, not single numbers.  

# 

# \## Don’t

# 

# \- Force users into rigid templates.  

# \- Hide assumptions behind calculations.
