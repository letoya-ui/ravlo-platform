# \# AI Analysis

# 

# AI Analysis synthesizes all deal inputs, comps, rehab, and financing into a clear, investor‑grade recommendation.

# 

# \## Purpose

# 

# This step answers:  

# “Is this a good deal, and why?”

# 

# It provides:

# \- risk assessment  

# \- ROI projections  

# \- sensitivity analysis  

# \- recommended next steps  

# \- deal strengths and weaknesses  

# 

# \## Anatomy

# 

# \- AI summary block  

# \- Risk score  

# \- ROI projections  

# \- Sensitivity sliders  

# \- Recommended actions  

# 

# \## Behavior

# 

# \- AI updates when any upstream data changes.  

# \- Insights are prioritized by impact.  

# 

# \## Do

# 

# \- Keep insights concise and actionable.  

# \- Tie every insight to a next step.  

# 

# \## Don’t

# 

# \- Overwhelm users with technical jargon.  

# \- Present conflicting recommendations.
