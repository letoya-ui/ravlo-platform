# \# Financing

# 

# Financing calculates loan structure, interest, payments, and total capital required.

# 

# \## Purpose

# 

# This step answers:  

# “How will I pay for this deal?”

# 

# It provides:

# \- loan options  

# \- interest rates  

# \- points and fees  

# \- monthly payments  

# \- total cash needed  

# \- lender‑specific requirements  

# 

# \## Anatomy

# 

# \- Loan type selector  

# \- Down payment input  

# \- Rate and term fields  

# \- Points and fees  

# \- Monthly payment breakdown  

# \- Total cash required  

# 

# \## Behavior

# 

# \- AI suggests optimal loan structures.  

# \- Users can compare multiple financing scenarios.  

# \- Totals update in real time.  

# 

# \## Do

# 

# \- Show all costs clearly.  

# \- Allow scenario comparison.  

# 

# \## Don’t

# 

# \- Hide lender fees.  

# \- Assume a single financing path.
