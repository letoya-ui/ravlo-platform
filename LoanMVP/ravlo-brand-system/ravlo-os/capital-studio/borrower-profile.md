# \# Borrower Profile

# 

# Borrower Profile captures the investor’s financial and experience background to determine lender fit and loan eligibility.

# 

# \## Purpose

# 

# This step answers:

# “Who is the borrower, and what is their financial profile?”

# 

# It provides:

# \- credit profile  

# \- experience level  

# \- liquidity  

# \- documentation requirements  

# \- lender eligibility  

# 

# \## Required Inputs

# 

# \- Credit score range  

# \- Experience level (deals completed)  

# \- Liquidity / reserves  

# \- Entity type (LLC, personal, partnership)  

# 

# \## Optional Inputs

# 

# \- Tax returns  

# \- Bank statements  

# \- W‑2 / 1099  

# \- P\&L statements  

# 

# \## Anatomy

# 

# \- Form fields  

# \- Experience badges  

# \- Liquidity indicators  

# \- Eligibility flags  

# 

# \## Behavior

# 

# \- Lender matching updates based on borrower profile.  

# \- AI highlights strengths and weaknesses.  

# 

# \## Do

# 

# \- Keep inputs simple and non‑intimidating.  

# \- Use ranges instead of exact numbers when possible.  

# 

# \## Don’t

# 

# \- Ask for unnecessary documents upfront.  

# \- Penalize new investors without explanation.
