# \# AI Capital Analysis

# 

# AI Capital Analysis synthesizes borrower profile, deal financials, and lender options into a clear financing recommendation.

# 

# \## Purpose

# 

# This step answers:

# “What is the smartest financing path for this deal?”

# 

# It provides:

# \- risk assessment  

# \- cost comparison  

# \- lender fit analysis  

# \- recommended loan structure  

# \- next steps  

# 

# \## Anatomy

# 

# \- AI summary block  

# \- Risk score  

# \- Cost comparison  

# \- Lender fit matrix  

# \- Recommended actions  

# 

# \## Behavior

# 

# \- AI updates when any upstream data changes.  

# \- Insights are prioritized by impact.  

# 

# \## Do

# 

# \- Keep insights concise and actionable.  

# \- Tie every insight to a next step.  

# 

# \## Don’t

# 

# \- Overwhelm users with technical lending jargon.  

# \- Present conflicting recommendations.

#
