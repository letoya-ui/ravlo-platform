# \# Documents \& Requirements

# 

# Documents \& Requirements outline what the lender needs to approve the loan.

# 

# \## Purpose

# 

# This step answers:

# “What documents do I need to provide?”

# 

# It provides:

# \- document checklist  

# \- upload areas  

# \- lender‑specific requirements  

# \- status indicators  

# 

# \## Common Documents

# 

# \- ID  

# \- Bank statements  

# \- Tax returns  

# \- Purchase contract  

# \- Rehab budget  

# \- Appraisal (later stage)  

# 

# \## Anatomy

# 

# \- Document list  

# \- Upload controls  

# \- Status badges (pending, uploaded, missing)  

# \- Notes  

# 

# \## Behavior

# 

# \- Requirements update based on lender selection.  

# \- Users can upload documents directly.  

# 

# \## Do

# 

# \- Keep the checklist simple and clear.  

# \- Use consistent status indicators.  

# 

# \## Don’t

# 

# \- Ask for documents irrelevant to the loan type.  

# \- Hide which documents are optional.
