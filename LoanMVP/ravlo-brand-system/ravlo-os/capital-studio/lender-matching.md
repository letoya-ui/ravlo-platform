# \# Lender Matching

# 

# Lender Matching connects the investor with lenders whose products fit the deal, borrower profile, and loan type.

# 

# \## Purpose

# 

# This step answers:

# “Which lenders are the best fit for this deal?”

# 

# It provides:

# \- matched lenders  

# \- rate and term ranges  

# \- eligibility indicators  

# \- required documents  

# \- next steps  

# 

# \## Anatomy

# 

# Each lender card includes:

# \- Lender name  

# \- Product type  

# \- Rate range  

# \- Term range  

# \- Points \& fees  

# \- Eligibility match score  

# \- Required documents  

# 

# \## Behavior

# 

# \- Matches update based on borrower profile and deal financials.  

# \- Users can compare lenders side‑by‑side.  

# \- AI highlights best‑fit lenders.  

# 

# \## Do

# 

# \- Keep lender cards clean and comparable.  

# \- Show eligibility clearly.  

# 

# \## Don’t

# 

# \- Hide lender requirements.  

# \- Overload users with too many options.
