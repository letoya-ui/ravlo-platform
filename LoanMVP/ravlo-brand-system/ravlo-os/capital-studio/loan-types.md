# \# Loan Types

# 

# Loan Types define the financing structure for the deal. They determine interest, points, fees, terms, and capital requirements.

# 

# \## Purpose

# 

# This step answers:

# “What type of loan is appropriate for this deal?”

# 

# It provides:

# \- loan type options  

# \- rate and term expectations  

# \- cost implications  

# \- lender availability  

# 

# \## Common Loan Types

# 

# \- Fix \& Flip Loan  

# \- DSCR Rental Loan  

# \- BRRRR Loan  

# \- Bridge Loan  

# \- Construction Loan  

# \- HELOC / Equity Loan  

# 

# \## Anatomy

# 

# Each loan type includes:

# \- Name  

# \- Description  

# \- Rate range  

# \- Term range  

# \- Points \& fees  

# \- Lender availability  

# 

# \## Behavior

# 

# \- Selecting a loan type updates deal financials.  

# \- AI suggests optimal loan types based on deal strategy.  

# 

# \## Do

# 

# \- Keep loan descriptions simple and investor‑friendly.  

# \- Show cost implications clearly.  

# 

# \## Don’t

# 

# \- Hide lender fees or points.  

# \- Mix loan types without clear distinctions.
