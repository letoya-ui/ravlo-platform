# \# Deal Financials

# 

# Deal Financials calculate the capital required, monthly payments, and total cost of financing.

# 

# \## Purpose

# 

# This step answers:

# “What will this loan cost, and how much cash do I need?”

# 

# It provides:

# \- down payment  

# \- loan amount  

# \- interest rate  

# \- monthly payments  

# \- points \& fees  

# \- total cash required  

# 

# \## Anatomy

# 

# \- Loan amount  

# \- Down payment  

# \- Rate \& term  

# \- Points \& fees  

# \- Monthly payment breakdown  

# \- Total cash required  

# 

# \## Behavior

# 

# \- Updates dynamically based on loan type and borrower profile.  

# \- AI suggests optimized structures (e.g., lower cash to close).  

# 

# \## Do

# 

# \- Show all costs clearly.  

# \- Use consistent formatting for currency and percentages.  

# 

# \## Don’t

# 

# \- Hide soft costs or lender fees.  

# \- Present unrealistic or overly precise estimates.

#
