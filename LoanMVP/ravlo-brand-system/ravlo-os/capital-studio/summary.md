# \# Capital Summary

# 

# Capital Summary provides a complete, investor‑ready overview of the financing structure, lender match, and capital requirements.

# 

# \## Purpose

# 

# This step answers:

# “What is the final financing package?”

# 

# It provides:

# \- loan structure  

# \- total cash required  

# \- monthly payments  

# \- lender selection  

# \- document status  

# \- AI recommendation  

# \- exportable capital package  

# 

# \## Anatomy

# 

# \- Summary metrics  

# \- Loan details  

# \- Lender selection  

# \- Document checklist  

# \- Recommended next steps  

# 

# \## Behavior

# 

# \- Summary updates dynamically as inputs change.  

# \- Users can export or share the capital package.  

# 

# \## Do

# 

# \- Keep the summary clean and scannable.  

# \- Highlight the most important metrics.  

# 

# \## Don’t

# 

# \- Overload the summary with raw data.  

# \- Hide risks or assumptions.
