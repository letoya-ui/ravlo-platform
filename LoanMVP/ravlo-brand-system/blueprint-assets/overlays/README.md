\# Blueprint Overlays



This folder contains blueprint-style overlays used to add depth and structure.



Use for:

\- Hero OS visuals

\- Ecosystem diagrams

\- Product modules

\- Background layering



Naming Convention:

ravlo-blueprint-overlay-\[pattern]-v1.\[format]



Patterns:

\- crosshatch

\- rings

\- grid-overlay

\- corner-brackets

\- measurement-marks



Notes:

\- Overlays should enhance depth without overpowering the UI.

\- Keep opacity low (10–25%).

\- Use Blueprint Blue or Soft Glow Blue for accents.
