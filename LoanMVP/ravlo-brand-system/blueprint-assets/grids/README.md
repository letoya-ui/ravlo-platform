\# Blueprint Grids



This folder contains architectural grid assets used across the Ravlo OS.



Use for:

\- Background structure

\- OS overlays

\- Section transitions

\- Technical diagrams



Naming Convention:

ravlo-blueprint-grid-\[density]-v1.\[format]



Examples:

\- ravlo-blueprint-grid-light-v1.png

\- ravlo-blueprint-grid-medium-v1.svg

\- ravlo-blueprint-grid-heavy-v1.png



Notes:

\- Keep line weight thin and precise.

\- Use Blueprint Blue (#3A5C7A) or Soft Glow Blue (#4C6E8C).

\- Avoid heavy opacity; grids should feel subtle and intentional.
