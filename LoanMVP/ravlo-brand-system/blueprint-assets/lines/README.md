\# Blueprint Lines



This folder contains individual blueprint line elements.



Use for:

\- UI accents

\- OS diagrams

\- Section dividers

\- Motion graphics



Naming Convention:

ravlo-blueprint-line-\[type]-v1.\[format]



Types:

\- straight

\- angled

\- dashed

\- dotted

\- connector



Notes:

\- Maintain consistent stroke width.

\- Glow should be soft, not neon.

\- Lines should scale cleanly for wide and tall compositions.
