\# Ecosystem Nodes



This folder contains individual node assets used in the ecosystem diagram.



Examples:

\- Deal Architect node

\- Renovation Studio node

\- Build Studio node

\- Capital Studio node

\- Portfolio Intelligence node

\- Partner Network node



Use for:

\- Interactive diagrams

\- Product pages

\- Hover states

\- UI animations



Naming Convention:

ravlo-node-\[module]-v1.\[format]



Notes:

\- Keep nodes consistent in size and glow intensity.

\- Use Blueprint Blue for accents.
