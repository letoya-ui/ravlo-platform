\# Blueprint Rings



This folder contains the circular blueprint rings used behind the ecosystem diagram.



Use for:

\- Background structure

\- OS depth

\- Motion graphics

\- Section transitions



Naming Convention:

ravlo-blueprint-ring-\[size]-v1.\[format]



Notes:

\- Use soft glow, not heavy neon.

\- Maintain thin, architectural line weight.

\- Rings should scale cleanly for wide and tall compositions.
