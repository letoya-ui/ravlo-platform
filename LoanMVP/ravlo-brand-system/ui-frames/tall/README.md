\# Tall UI Frames



This folder contains tall-format UI compositions for Ravlo.



Use for:

\- Mobile previews

\- Vertical storytelling

\- Social assets

\- Product detail pages



Naming Convention:

ravlo-uiframe-tall-\[module]-v1.\[format]



Notes:

\- Maintain 9:16 or 4:5 aspect ratios.

\- Keep glow subtle and controlled.

\- Use spotlight gradients for depth.
