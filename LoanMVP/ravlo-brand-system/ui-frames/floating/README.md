\# Floating UI Frames



This folder contains floating panel compositions.



Use for:

\- OS previews

\- Dashboard highlights

\- Interactive product sections

\- Motion graphics



Naming Convention:

ravlo-uiframe-floating-\[panel]-v1.\[format]



Notes:

\- Panels should appear weightless but grounded.

\- Use soft shadows and blueprint accents.

\- Maintain consistent spacing and elevation.
