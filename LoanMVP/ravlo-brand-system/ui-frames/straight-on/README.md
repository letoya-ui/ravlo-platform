\# Straight-On UI Frames



This folder contains straight-on, flat UI compositions.



Use for:

\- Documentation

\- Product clarity shots

\- Feature explanations

\- Clean UI presentations



Naming Convention:

ravlo-uiframe-straighton-\[module]-v1.\[format]



Notes:

\- No perspective or tilt.

\- Keep everything symmetrical and balanced.

\- Use matte steel backgrounds for clarity.
