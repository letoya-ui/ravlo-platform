\# Angled UI Frames



This folder contains angled UI compositions for dynamic storytelling.



Use for:

\- Cinematic transitions

\- Product reveals

\- Section intros

\- Motion sequences



Naming Convention:

ravlo-uiframe-angled-\[module]-v1.\[format]



Notes:

\- Use subtle perspective, not extreme tilt.

\- Maintain clean geometry and alignment.

\- Keep glow minimal to avoid distortion.
