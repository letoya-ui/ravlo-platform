\# Wide UI Frames



This folder contains wide-format UI compositions for Ravlo.



Use for:

\- Homepage sections

\- Product modules

\- Tour pages

\- Hero OS expansions



Naming Convention:

ravlo-uiframe-wide-\[module]-v1.\[format]



Notes:

\- Maintain cinematic 21:9 or 16:9 aspect ratios.

\- Use matte bluish-gray base with soft blueprint glow.

\- Keep depth layering consistent across modules.
