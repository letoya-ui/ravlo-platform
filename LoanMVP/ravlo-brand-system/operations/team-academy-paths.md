# Team Academy Paths

Ravlo team members should have learning paths that match their responsibilities.

## Operations Lead

Recommended Academy path: Lending

Focus areas:

- loan process
- borrower documentation
- lending operations
- CRM follow-up
- onboarding support
- processor and underwriting workflow awareness

## Construction Partner

Recommended Academy path: Contractor

Focus areas:

- rehab scopes
- repair estimates
- renovation budgets
- Build Studio workflow
- project timelines
- contractor coordination
- investor project support

## Dashboard expectation

The Operations Center should link the Operations Lead to the Lending Academy path.

The Partner Network dashboard should link the Construction Partner to the Contractor Academy path.

## Future routing rule

Add email-based Academy track overrides so internal team members can be assigned to the right learning path even if their technical role is broader than the learning path.
