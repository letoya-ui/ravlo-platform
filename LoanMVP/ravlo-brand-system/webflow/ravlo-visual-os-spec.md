# Ravlo Visual OS Spec
The cinematic identity of the Ravlo universe — atmosphere, lighting, blueprint geometry, and depth.

---

# 1. Ravlo Atmosphere Layer

The atmospheric foundation of the OS.

### Primary Atmosphere
- Midnight gradient: deep navy ? near-black  
- Subtle noise texture (1–2%)  
- Soft radial spotlight behind content  
- Slight atmospheric haze (2–4%)  

### Mood
- Calm  
- Cinematic  
- Intentional  
- Premium  
- Architectural  

### Rules
- Never flat black  
- Never pure white  
- Always depth and dimension  

---

# 2. Ravlo Blueprint Geometry

The architectural backbone of the OS.

### Blueprint Elements
- Circular blueprint rings  
- Radial grid lines  
- Thin architectural crosshairs  
- Soft glowing nodes  
- Subtle dashed orbit lines  

### Usage
- Behind hero content  
- Behind panels  
- Behind module headers  
- Behind ecosystem diagrams  

### Rules
- Blueprint lines: opacity 6–12%  
- Rings scale with viewport  
- Never distract from content  
- Always feel engineered  

---

# 3. Ravlo Lighting System

Lighting defines the cinematic presence.

### Lighting Types
- Spotlight: soft radial glow  
- Rim Light: thin edge glow  
- Ambient Glow: subtle color wash  
- Depth Shadow: long, soft atmospheric shadow  

### Lighting Colors
- Steel blue  
- Midnight navy  
- Soft white  
- Blueprint cyan  

### Rules
- Light guides attention  
- Shadows must be soft  
- Panels should feel lit, not flat  

---

# 4. Ravlo Panel System

Panels are the core building block of the Visual OS.

### Panel Types
- Floating Panel  
- Glass Panel  
- Blueprint Panel  

### Panel Anatomy
- Panel container  
- Panel header  
- Panel body  
- Panel glow  
- Panel shadow  

### Panel Properties
- Rounded corners: 12–16px  
- Padding: 32–48px  
- Depth shadow: 20–40px blur  
- Glow: 4–8px soft edge  

### Rules
- Panels must feel like objects in space  
- Panels should never touch viewport edges  
- Panels float above the atmosphere  

---

# 5. Ravlo Card System

Cards are smaller, lighter versions of panels.

### Card Types
- Metric card  
- Feature card  
- Module card  
- Partner card  

### Card Properties
- Padding: 24–32px  
- Soft glow: 2–4px  
- Subtle border: 1px blueprint cyan (6% opacity)  
- Depth shadow: 10–20px blur  

### Rules
- Cards should feel premium  
- No harsh borders  
- Maintain breathing room  

---

# 6. Ravlo Depth System

Depth creates the cinematic layering.

### Depth Layers
- Layer 0: Atmosphere  
- Layer 1: Blueprint geometry  
- Layer 2: Panels  
- Layer 3: Cards  
- Layer 4: Floating UI elements  
- Layer 5: Hero content  

### Rules
- Higher layers = stronger glow + shadow  
- Lower layers = softer, atmospheric  
- Never flatten layers  

---

# 7. Ravlo Motion System

Motion is slow, calm, intentional.

### Motion Types
- Fade-in (200–400ms)  
- Slide-up (20–40px)  
- Soft scale (1–2%)  
- Glow pulse (4–6s cycle)  
- Blueprint drift (20–30s cycle)  

### Rules
- Motion should feel like breathing  
- Never fast or bouncy  
- Always cinematic  

---

# 8. Ravlo Color System (Visual OS Layer)

### Core Colors
- Midnight Black (#0A0D12)  
- Deep Navy (#0F1621)  
- Steel Blue (#1C2A3A)  
- Blueprint Cyan (#4CC3FF)  
- Soft White (#F2F5F8)  

### Accent Colors
- Capital Gold  
- Rehab Green  
- Build Studio Copper  
- Deal Architect Blue  
- Portfolio Purple  

### Rules
- Accents used sparingly  
- Blueprint cyan is primary glow color  
- Avoid saturated large areas  

---

# 9. Ravlo Visual Principles

1. Cinematic, not flashy  
2. Architectural, not decorative  
3. Calm, not sterile  
4. Dimensional, not flat  
5. Modular, not chaotic
