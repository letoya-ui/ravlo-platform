# Ravlo Dashboard UI System
The architectural foundation of the Ravlo investor experience — panels, metrics, grids, navigation, and workflow surfaces.

---

# 1. Dashboard Philosophy

The Ravlo dashboard is:
- A cinematic command center  
- Calm, structured, and investor-grade  
- Built around clarity and decision-making  
- Modular and scalable  
- Designed like an operating system, not a website  

Every element must feel intentional and engineered.

---

# 2. Dashboard Layout Structure

### Core Layout Regions
- **Primary Navigation** (left or top)  
- **Command Header** (top bar)  
- **Main Workspace** (panels + grids)  
- **Context Sidebar** (optional)  
- **Footer / System Status** (optional)  

### Grid System
- ravlo-grid-2  
- ravlo-grid-3  
- ravlo-grid-4  
- ravlo-grid-auto  

### Spacing
- Section padding: space-48 to space-96  
- Panel padding: space-32  
- Card padding: space-24

# 4. Metric System

Metrics are the heartbeat of the dashboard.

### Metric Types
- Primary Metric  
- Secondary Metric  
- Trend Metric  
- Comparison Metric  

### Metric Anatomy
- Label  
- Value

### Metric Rules
