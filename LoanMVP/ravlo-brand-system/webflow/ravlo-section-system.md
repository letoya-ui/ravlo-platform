# Ravlo Section System
The reusable page-level building blocks that structure the Ravlo OS and marketing site — heroes, features, pricing, testimonials, CTAs, and more.

---

# 1. Section Philosophy

Ravlo sections are:
- Cinematic and architectural  
- Calm and intentional  
- Structured for clarity  
- Designed to guide the eye  
- Consistent across all surfaces  

Sections are the macro-level containers that shape the Ravlo experience.

# 4. Section Spacing Tokens

### Vertical Spacing
- space-64 (default)  
- space-80 (hero)  
- space-96 (cinematic sections)  

### Horizontal Spacing
- Max width: 1200–1320px  
- Side padding: space-24 to space-48
