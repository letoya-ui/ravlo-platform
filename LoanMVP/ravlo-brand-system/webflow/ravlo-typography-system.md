# Ravlo Typography System
The voice of the Ravlo OS — calm, cinematic, architectural, and investor-grade.

---

# 1. Typography Philosophy

Ravlo typography is:
- Clean and modern  
- Highly readable  
- Calm and confident  
- Structured like architecture  
- Designed for investor decision-making  

Typography should never feel loud or playful — it should feel engineered.

---

# 2. Font Families

### Primary Typeface
**Inter** (or any modern geometric sans-serif)

Weights used:
- 900 — Extra Bold  
- 700 — Bold  
- 600 — Semi Bold  
- 500 — Medium  
- 400 — Regular  

### Secondary Typeface (Optional)
**IBM Plex Sans** or **Space Grotesk**  
Used for technical or blueprint-style moments.

---

# 3. Heading Scale

### heading-xl
- Size: 48–64px  
- Weight: 700  
- Line height: 110%  
- Usage: Hero titles, major OS moments  

### heading-lg
- Size: 36–48px  
- Weight: 700  
- Line height: 115%  
- Usage: Section headers, module intros  

### heading-md
- Size: 28–32px  
- Weight: 600  
- Line height: 120%  
- Usage: Card titles, panel headers  

### heading-sm
- Size: 20–24px  
- Weight: 600  
- Line height: 130%  
- Usage: Subheaders, feature titles  

---

# 4. Body Text

### body-lg
- Size: 20px  
- Weight: 400–500  
- Line height: 150%  
- Usage: Hero subtext, important descriptions  

### body-md
- Size: 16–18px  
- Weight: 400  
- Line height: 150%  
- Usage: Standard reading text  

### body-sm
- Size: 14–15px  
- Weight: 400  
- Line height: 150%  
- Usage: Card descriptions, metadata  

---

# 5. Labels & Utility Text

### label
- Size: 12–13px  
- Weight: 600  
- Letter spacing: +2%  
- Usage: Tags, categories, UI labels  

### label--accent
- Same as label  
- Color: Blueprint Cyan  
- Usage: Technical or blueprint moments  

---

# 6. Responsive Type Rules

### Desktop XL (1440px+)
- Full scale  
- Generous spacing  

### Desktop (1200px)
- Slight reduction in heading-xl and heading-lg  

### Tablet (992px)
- heading-xl ? heading-lg  
- heading-lg ? heading-md  

### Mobile (768px and below)
- All headings scale down 1–2 steps  
- Body text remains 16px minimum  

Typography must remain readable at all sizes.

---

# 7. Line Height Rules

- Headings: 110–130%  
- Body text: 150%  
- Labels: 120–130%  

Ravlo uses **tight headings** and **comfortable body text**.

---

# 8. Color Usage

### Headings
- Soft White (#F2F5F8)

### Body Text
- Soft White at 80–90% opacity

### Labels
- Blueprint Cyan or Soft White at 60–80% opacity

Typography should always contrast clearly against the midnight atmosphere.

---

# 9. Do & Don't Examples

### Do
- Use consistent hierarchy  
- Keep headings tight  
- Maintain calm spacing  
- Use labels sparingly  

### Don't
- Use overly decorative fonts  
- Mix too many weights  
- Use bright saturated colors for text  
- Overuse uppercase  

---

# 10. Typography Principles

1. Clarity over decoration  
2. Calm over loud  
3. Structure over chaos  
4. Consistency builds trust  
5. Typography is the voice of the OS
