\# Spotlight Backgrounds



This folder contains soft spotlight gradients for cinematic depth.



Use for:

\- Hero OS

\- Section transitions

\- Product highlights

\- Visual storytelling



Naming Convention:

ravlo-bg-spotlight-\[wide|tall]-v1.\[format]



Notes:

\- Use soft radial gradients.

\- Keep glow subtle and controlled.

\- Avoid harsh edges or bright centers.
