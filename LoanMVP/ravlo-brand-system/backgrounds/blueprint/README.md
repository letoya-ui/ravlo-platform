\# Blueprint Backgrounds



This folder contains blueprint-style backgrounds with architectural lines.



Use for:

\- OS overlays

\- Ecosystem sections

\- Technical pages

\- Motion graphics



Naming Convention:

ravlo-bg-blueprint-\[grid|lines|overlay]-v1.\[format]



Notes:

\- Use Blueprint Blue (#3A5C7A) for accents.

\- Keep line weight thin and architectural.

\- Glow should be soft, not neon.
