\# Midnight Backgrounds



This folder contains Ravlo's primary dark backgrounds.



Use for:

\- Hero sections

\- OS visuals

\- Product pages

\- Dashboard foundations



Naming Convention:

ravlo-bg-midnight-\[wide|tall]-v1.\[format]



Notes:

\- Base color: #0C1116

\- Keep gradients subtle and cinematic.

\- Avoid heavy noise or texture.
