\# Steel Backgrounds



This folder contains matte bluish-gray steel textures.



Use for:

\- UI panels

\- Monogram base

\- Section dividers

\- Product modules



Naming Convention:

ravlo-bg-steel-\[wide|tall]-v1.\[format]



Notes:

\- Base tone: #6B7F93

\- Keep finish matte with minimal shine.

\- Avoid metallic reflections; Ravlo is grounded, not glossy.
